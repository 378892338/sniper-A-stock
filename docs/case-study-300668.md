# 杰恩设计 300668 平台内涨停案例分析

## 时间范围
2025-09-01 至 2026-04-30

## 走势拆解

### 第一阶段：低位平台 + 平台内涨停（吸筹）
- **09-01 ~ 09-02**: 17.36 ~ 17.90 底部窄幅震荡
- **09-03**: 17.50 → 20.60，第一根20cm涨停，收盘=涨停价，封死
  - 成交额 28351万，量能放大
- **09-04 ~ 12-19**: 涨停后横盘约3.5个月，核心区间 17.20 ~ 20.02
  - 涨停次日巨量换手（成交额 11.5亿）
  - 股价不突破，继续缩量整理
  - 这是典型的"平台内涨停"吸筹模式

### 第二阶段：中阳突破 + 加速（启动）
- **12-22**: 19.88 → 20.90，突破平台上沿 20.60，一根中阳线
- **12-25**: 22.00 → 26.60，第二根20cm涨停，突破确认后加速
- **12-26 ~ 01-08**: 继续拉升到 31.25

### 第三阶段：回调洗盘
- **01-09 ~ 02-13**: 31 → 24.5，回落超20%，量能萎缩

### 第四阶段：主升浪
- **03-23 ~ 04-14**: 29.94 → 57.22，12个交易日近乎翻倍
- **04-30**: 高点 66.96，冲高回落收 55.29

## 涨幅
从第一根涨停 17.50 到最后高点 66.96，涨幅 +283%

## 模式总结
1. 长期横盘（3个月+）
2. 涨停发生在平台内部（吸筹信号）
3. 涨停后继续横盘不突破（洗盘）
4. 中阳线突破平台上沿（启动信号=买点）
5. 横有多长竖有多高
