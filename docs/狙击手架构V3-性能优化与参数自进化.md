## 2. 极限性能优化方案

### 2.1 架构改造：三层数据缓存

```
┌─────────────────────────────────────────────────────────┐
│  Layer 0: In-Memory Cache (内存)                         │
│  - Daily bars: dict[symbol] → DataFrame, loaded at startup │
│  - L0/L1 scores: dict[date] → score, cached after first calc |
│  - L2 factors: dict[date] → DataFrame(all_stocks), precomputed |
├─────────────────────────────────────────────────────────┤
│  Layer 1: Parquet Cache (磁盘)                           │
│  - 预计算结果: l0_scores_{start}_{end}.parquet          │
│  - 预计算结果: l1_sector_scores_{start}_{end}.parquet   │
│  - 预计算结果: l2_factors_{date}.parquet                │
│  - 回测结果: backtest_result_{params_hash}.parquet      │
├─────────────────────────────────────────────────────────┤
│  Layer 2: SQLite Index (元数据)                          │
│  - 参数-结果映射: params → metric_summary               │
│  - 参数优化历史: 所有组合的 Sharpe/回撤/胜率             │
│  - 因子有效性: IC/IR 历史记录                           │
└─────────────────────────────────────────────────────────┘
```

**缓存策略**:

| 数据类型 | 缓存位置 | TTL | 更新频率 |
|---------|---------|-----|---------|
| 日线行情 | 启动时一次性加载 | 回测期间不变 | 数据更新时重建 |
| L0 评分 | 内存 dict[date→score] | 回测期间不变 | 回测前预计算 |
| L1 板块评分 | 内存 dict[date→DataFrame] | 回测期间不变 | 回测前预计算 |
| L2 因子 | 磁盘 parquet(date→DataFrame) | 周频 | 每周五收盘后 |
| L2 评分 | 内存 dict[date→DataFrame] | 回测期间不变 | 从 L2 因子计算 |
| 回测结果 | 磁盘 parquet(params_hash→metrics) | 永久 | 首次计算 |
| 参数-结果索引 | SQLite params_results.db | 永久 | 每次优化追加 |

### 2.2 L2 选股向量化改造（最大瓶颈）

**当前问题**: `top_stocks()` 调用 `_preload_bars()` 加载全市场 → 逐只评分

**改造方案**:

```python
class StockScorer:
    def top_stocks_vectorized(self, date: str, sectors: list[str],
                               factor_cache: dict[str, pd.DataFrame]) -> list[dict]:
        """向量化 L2 选股：批量计算所有股票的 12 因子，一次性排序。
        
        性能提升: 15分钟 → 30秒 (×30)
        """
        # 1. 从缓存加载当日所有活跃股票 DataFrame
        all_stocks_df = factor_cache[date]  # 预计算的 12 因子值
        
        # 2. 从缓存加载板块映射
        sector_map = self._get_sector_map(date)  # dict[symbol → sector]
        
        # 3. 过滤: 只在指定板块内
        if sectors:
            all_stocks_df = all_stocks_df[all_stocks_df['sector'].isin(sectors)]
        
        # 4. 应用 L3 硬性条件过滤 (向量化)
        mask = self._hard_filter_vectorized(all_stocks_df, date)
        all_stocks_df = all_stocks_df[mask]
        
        # 5. 加权合成评分 (向量化)
        weights = np.array([
            CFG.trend_factor_weight, CFG.volume_factor_weight,
            CFG.macd_factor_weight, CFG.rsi_factor_weight,
            CFG.fund_flow_weight, CFG.big_order_weight,
            CFG.dragon_tiger_weight, CFG.eps_weight,
            CFG.roe_weight, CFG.revenue_growth_weight,
            CFG.market_cap_weight, CFG.turnover_weight,
        ])
        factor_cols = ['trend', 'volume', 'macd', 'rsi', 'fund_flow', 'big_order',
                       'dragon_tiger', 'eps_score', 'roe_score', 'revenue_growth',
                       'market_cap', 'turnover_score']
        all_stocks_df['composite'] = all_stocks_df[factor_cols].values @ weights
        
        # 6. 排序 + 柔性条件过滤
        all_stocks_df = all_stocks_df.sort_values('composite', ascending=False)
        all_stocks_df = all_stocks_df[all_stocks_df['composite'] >= CFG.soft_min_score]
        if sectors:
            all_stocks_df = all_stocks_df[all_stocks_df['sector_rank'] <= CFG.soft_sector_top]
        
        # 7. 返回 Top N
        top = all_stocks_df.head(CFG.top_n)
        return top.to_dict('records')
```

| L2 评分耗时/天 | ~8分钟 | ~0.1秒 | **>4000x** |

### 2.4 回测结果缓存 + 参数哈希

**当前问题**: 每次回测从头跑，无法复用

**改造方案**:

```python
import hashlib
import json

def _params_hash(params: dict) -> str:
    """参数组生成唯一哈希。"""
    params_str = json.dumps(params, sort_keys=True)
    return hashlib.md5(params_str.encode()).hexdigest()[:8]

class CachedBacktestEngine:
    def run(self, params: dict, start_date="", end_date="") -> dict:
        phash = _params_hash(params)
        cache_key = f"backtest_{phash}_{start_date}_{end_date}"
        
        # 检查缓存
        cached = self._cache_lookup(cache_key)
        if cached is not None:
            return cached
        
        # 运行回测
        result = self._run_inner(params, start_date, end_date)
        
        # 写入缓存
        self._cache_save(cache_key, result)
        return result
```

### 2.5 参数优化并行化

**当前问题**: 串行执行，27组合串行耗时 = 27 × 单回测时间

```python
from concurrent.futures import ProcessPoolExecutor, as_completed

def run_single_param(params: dict, start_date: str, end_date: str) -> dict:
    """在独立进程中运行单次回测（避免配置共享问题）。"""
    # 重建配置（隔离进程）
    import sniper.config as cfg
    # 应用参数覆盖...
    engine = BacktestEngine()
    return engine.run(start_date, end_date)

def optimize_parallel(param_grid: dict, n_workers: int = None) -> list[dict]:
    """并行参数优化。"""
    import itertools
    import multiprocessing
    
    keys = list(param_grid.keys())
    param_combos = [
        dict(zip(keys, vals)) 
        for vals in itertools.product(*param_grid.values())
    ]
    
    n_workers = n_workers or min(multiprocessing.cpu_count(), 8)
    
    results = []
    with ProcessPoolExecutor(max_workers=n_workers) as executor:
        futures = {
            executor.submit(run_single_param, params, "2019-01-01", "2026-05-08"): params
            for params in param_combos
        }
        for i, future in enumerate(as_completed(futures), 1):
            params = futures[future]
            try:
                result = future.result()
                results.append(result)
                logger.info(f"[{i}/{len(param_combos)}] {params} → Sharpe={result.get('sharpe', 0):.2f}")
            except Exception as e:
                logger.warning(f"{params} 失败: {e}")
    
    return results
```

```python
@classmethod
def from_consolidated_cache(cls, cache_dir) -> "DataStore":
    """从 consolidated daily_bars.parquet 加载"""
    df = pd.read_parquet(cache_dir / "daily_bars.parquet")
    store = cls()
    for sym, grp in df.groupby("symbol"):
        store.add_daily(sym, grp.set_index("date").sort_index())
    return store
```

#### 4.2.3 参数管理器（Param Manager）

```python
class ParamManager:
    """实盘参数管理器：存储 + 更新 + 回滚。"""
    
    def __init__(self, db_path: str = "params.db"):
        self.db = ParamDB(db_path)
        self._current_params = self._load_latest()
        self._history: list[dict] = []  # 参数变更历史
    
    def get_current(self) -> dict:
        """获取当前生效参数。"""
        return self._current_params.copy()
    
    def apply_suggestion(self, suggestion: dict, min_improvement: float = 0.01):
        """应用评估建议，需满足最小改进阈值。"""
        new_params = self._current_params.copy()
        changed = False
        
        for key, info in suggestion.items():
            if info["impact"] >= min_improvement and info["confidence"] > 0.5:
                old_val = new_params.get(key)
                new_params[key] = info["optimal"]
                changed = True
                logger.info(f"[参数更新] {key}: {old_val} → {info['optimal']} "
                           f"(影响={info['impact']:.3f}, 置信={info['confidence']:.2f})")
        
        if changed:
            self._history.append({
                "date": datetime.now().strftime("%Y-%m-%d"),
                "old": self._current_params.copy(),
                "new": new_params.copy(),
                "trigger": "monthly_calibration",
            })
            self._current_params = new_params
            self._save(new_params)
            self._save_history()
    
    def rollback(self, n_steps: int = 1):
        """回滚参数（回测验证失败时手动/自动回滚）。"""
        if len(self._history) >= n_steps:
            old = self._history[-n_steps]["old"]
            self._current_params = old.copy()
            self._save(old)
            logger.warning(f"[参数回滚] 回滚 {n_steps} 步")
    
    def get_history(self) -> list[dict]:
        return self._history
```

#### 4.2.4 在线学习（Online Learning）

```python
class OnlineParamLearner:
    """
    在线学习参数更新（可选，替代月频校准）。
    
    原理: 每次交易后，根据该交易的 PnL 和当时市场环境，
    调整参数权重。使用带权重的线性回归估计参数敏感性。
    """
    
    def __init__(self, param_keys: list[str]):
        self.param_keys = param_keys
        self.history: list[tuple[dict, float]] = []  # (params_at_trade, pnl)
        self.coefficients: dict[str, float] = {k: 0.0 for k in param_keys}
    
    def observe(self, params: dict, pnl: float):
        """记录一次观测。"""
        self.history.append((params, pnl))
        self._update_coefficients()
    
    def _update_coefficients(self):
        """增量更新参数敏感性系数。"""
        if len(self.history) < 10:
            return
        # 最近 100 笔交易
        recent = self.history[-100:]
        
        X = np.array([list(p.items()) for p, _ in recent])
        y = np.array([pnl for _, pnl in recent])
        
        # 简单线性回归: pnl = β0 + Σ βi × param_i
        coeffs, _, _, _ = np.linalg.lstsq(X, y, rcond=None)
        self.coefficients = dict(zip(self.param_keys, coeffs))
    
    def suggest(self) -> dict:
        """基于系数方向建议参数调整。"""
        suggestions = {}
        for key, beta in self.coefficients.items():
            # 正系数 → 增大参数有利，负系数 → 减小有利
            suggestions[key] = {
                "sensitivity": beta,
                "direction": "increase" if beta > 0.01 else "decrease" if beta < -0.01 else "stable",
            }
        return suggestions
```

### 4.3 参数更新安全机制

| 机制 | 说明 | 触发条件 |
|------|------|---------|
| **最小改进阈值** | 建议参数变更必须影响 >1% | 月频校准自动应用 |
| **置信度要求** | 参数建议需 confidence >0.5 | 同上 |
| **参数变更频率限制** | 单次变更间隔 ≥ 30 天 | 防止过度调参 |
| **参数漂移检测** | 参数偏离初始值 >20% 时告警 | 月频检查 |

| **参数范围约束** | 每个参数有硬范围，不可超出 | 始终生效 |
| **A/B 并行测试** | 新参数以 50% 仓位并行运行 | 可选 |

### 4.4 参数漂移检测

```python
class ParamDriftDetector:
    """检测参数是否过度偏离初始值。"""
    
    def __init__(self, initial_params: dict, threshold: float = 0.20):
        self.initial = initial_params
        self.threshold = threshold
    
    def check(self, current_params: dict) -> dict:
        """检查每个参数的漂移程度。"""
        drifts = {}
        for key, val in current_params.items():
            initial = self.initial.get(key, val)
            if isinstance(initial, float) and initial != 0:
                drift = abs(val - initial) / abs(initial)
            else:
                drift = 0.0 if val == initial else 1.0
            
            drifts[key] = {
                "initial": initial,
                "current": val,
                "drift_pct": drift,
                "flag": drift > self.threshold,
            }
        return drifts
```
