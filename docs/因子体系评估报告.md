## 第二部分：问题诊断

### 2.1 信息浪费

| 浪费 | 详情 | 级别 |
|------|------|------|
| **11个因子坍缩为1个总分** | 多因子评分→加权→聚合为一个score，买卖决策只看score，各因子独立信息全丢失 | 严重 |
| **3个子维度评分被丢弃** | `trend_score/alpha_score/risk_score` 计算后未写入 `month_scores` | 严重 |
| **5种形态识别全部跳过** | `fast_mode=True` → `pattern_mult`恒为1.0 | 严重 |
| **precompute.py 不被回测使用** | 回测另起炉灶按月重算11个因子 | 中 |
| **流动性过滤未接入** | 函数存在但 `run_pre_filter()` 未调用 | 中 |

### 2.2 因子设计方向错误

| 问题 | 层级 | 详情 |
|------|------|------|
| 月线DIF<0误用作看空 | L1 | DIF<0但上升实际看多，6-12个月恢复期被持续误判 |
| 创业板一票否决权重过高 | L1 | 单一市场技术信号压制全市场 |
| B3周线底背驰结构性不可能 | L2 | 牛市中价格做更高低点，永不触发 |
| 门控阈值方向反了 | L2 | 牛市3/5最严、偏弱1/5最松 |
| 看空阈值方向反了 | L2 | 牛市3最敏感、偏弱5最迟钝 |
| L3看空阈值固定 | L3 | 固定5不分市场，牛市正常回调易误拦 |
| 退出信号无牛市阻尼 | L3 | 6个信号牛熊标准完全一致 |
| B4窗口仅4周 | L2 | 过短，噪声大 |

### 2.3 因子体系结构缺陷

| 缺陷 | 详情 |
|------|------|
| **纯量价，无基本面** | 估值(PE/PB)、成长(ROE/利润增速)、质量因子完全缺失 |
| **资金面深度不足** | 仅有北向净流量的简单判断，缺少北向分行业、大单流向、融资融券等深度分析 |
| **无市场广度因子** | 涨跌比、创新高新低比、均线以上占比等常用市场状态判断因子缺失 |
| **无RSRS择时** | 机构量化广泛使用的支撑阻力相对强度择时未接入 |
| **无拥挤度因子** | 板块/个股交易拥挤度监测缺失，无法预警过热 |
| **无截面相对因子** | 没有因子值在全体股票中的百分位排名，因子值不可比 |

## 第四部分：因子协同优化方案

### 4.1 设计原则

**原则一：信息保全** — 当前系统将11个因子坍缩为1个分数，子维度信息全丢。必须保留多维度评分到决策端。

**原则二：状态依赖** — 因子有效性随市场状态变化。动量在牛市有效、反转在震荡有效、低波动在熊市有效。同一因子在不同状态下应分配不同权重。

**原则三：因子舒适区** — 每个因子在特定股票子集中表现最好。不应要求一个因子对所有股票都有效。

### 4.2 三层因子架构（建议）

```
L1: 市场状态判定
  ├── MACD多周期共振（现有，需修正DIF逻辑）
  ├── RSRS择时（新增，替代部分MACD判断）
  ├── 市场广度（新增，涨跌比/均线以上占比）
  ├── 资金面（现有，深化北向+融资）
  └── 阴跌检测（现有）
  → 输出：市场状态 + 置信度 + 仓位建议

L2: 板块选择 + 评分
  ├── 趋势因子（现有Trend 5dim，修正门控方向）
  ├── 资金流（深化：北向分行业 + 大单）
  ├── 板块动量（新增：4周/13周收益排名）
  ├── 拥挤度（新增：换手率分位数）
  └── 形态评分（现有Pattern Bonus，修复跳过bug）
  → 输出：板块评分（连续0-100，而非二元通过/不通过）

L3: 个股三维评分
  ├── Trend（现有：trend_macd_5dim + momentum_12_1）
  ├── Alpha（现有：reversal_20 + bottom_divergence + 形态乘数）
  ├── Risk（现有：low_volatility + turnover_stability）
  ├── Value（新增：PE/PB百分位）
  └── Fund（新增：北向持仓+DDE大单）
  → 输出：四维评分 + 综合分，全部保留到决策端

决策端:
  ├── 买入：综合分排序 × 评分门槛(≥48) × 板块权重
  ├── 持仓：四维评分加权（而非等权）
  ├── 卖出：止损(-15%) + 时间止损(30日+浮亏>5%) + 信号退出
  └── 震荡市：板块轮动模式（2-4周换仓）
```

### 4.3 具体改进步骤

#### 第1步：因子评价（IC/IR/分组回测）

对现有所有因子运行定量评价：

- L3的11个因子：按月频计算Rank IC、ICIR、5分位分组收益
- L1/L2的门控条件：作为事件型因子，评价命中后的收益分布
- 各因子分域（牛市/震荡/熊市）表现

**产出**：每个因子的IC时间序列 + ICIR + 分组单调性 + 分域有效性表格

#### 第2步：因子修正

根据评价结果：

| 优先级 | 修改内容 | 预期效果 |
|--------|---------|---------|
| P0 | 月线DIF<0→DIF<0且下降、L2门控反转、L2看空反转、创业板否决分级、仓位调整 | 修复方向错误，牛市识别率9.8%→25-30% |
| P1 | L3三维评分保留到决策端、形态乘数生效、MIN_SCORE门槛、止损、L3看空动态化 | 信息保全，夏普0.226→0.35-0.45 |

#### 第3步：因子扩展

| 批次 | 新增因子 | 接入成本 | 预期增益 |
|------|---------|---------|---------|
| 第一批 | RSRS择时(L1) + PE/PB百分位(L3) + 北向分行业(L2) | 低中 | +1-2%年化 |
| 第二批 | 市场广度(L1) + DDE资金流(L3) + 拥挤度(L2) | 中 | +0.5-1.5%年化 |
