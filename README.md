Sniper狙击手 A 股选股与回测系统（Python 3.13）

 基于 L0-L4 四层选股架构 + 打字机归因自进化回测引擎

 ## 核心特性
 - L0-L4 四层选股（市场状态 → 板块 → 个股 → 入场 → 退出）
 - 打字机归因自进化回测
 - 多数据源自动降级（免费/付费混合）
 - 量化日报（Markdown / HTML / PDF）

 ## 技术栈
 Python 3.13 · pandas · numpy · akshare · tushare · pytest

# 📈 股票量化分析与自动推送工具

本项目内置了完整的量化分析引擎，支持**归因分析、回归测试、条件参数因子自动筛选**以及**电脑端日报自动推送**等核心功能。

## ⚠️ 数据源特别说明

由于目前主流公开股票数据接口普遍存在严格的限流策略，导致数据源极不稳定。
为了保证程序的正常运行，**本项目内置任何默认数据源都不稳定**。如需稳定运行，请懂行的朋友自行配置可用的数据源接口。
