# 005 回测引擎未考虑交易成本、滑点和印花税

**严重程度**: HIGH  
**文件**: `backtest/run.py:103-176`  
**类型**: 功能缺失

## 问题描述

`BacktestEngine.run` 方法中：
- 没有扣除交易佣金（通常万 2.5）
- 没有扣除印花税（A 股卖出 0.1%）
- 没有模拟滑点
- 月度调仓时假设能以当日收盘价成交

对于月度调仓策略，年化换手约为 12 次（每次全仓换），不考虑成本会严重高估收益。以每次换手成本 0.15%（佣金+印花税+滑点），年化高估约 1.8%。

## 修复建议

```python
TRANSACTION_COST = 0.0015  # 单边 0.15%

# 在调仓时扣除
for date in pd.date_range(start_hold, next_date, freq="B"):
    ...
    if date == start_hold:  # 调仓日扣除成本
        turnover_cost = sum(abs(wt) for wt in weights.values()) * TRANSACTION_COST
        portfolio_returns.loc[date] = sum(rets) - turnover_cost
```

## 影响范围

回测的年化收益、夏普比率等指标全部偏高。
