# 011 项目完全没有单元测试

**严重程度**: LOW  
**文件**: 整个项目  
**类型**: 工程实践

## 问题描述

项目中没有任何测试文件（`test_*.py` 或 `*_test.py`）。

虽然项目规范要求 80% 测试覆盖率，但当前覆盖率为 0%。

## 建议

至少为核心模块添加测试：

| 模块 | 建议测试内容 |
|------|-------------|
| `models/factors.py` | 均线计算、MACD、涨停判定、各形态检测 |
| `models/sector.py` | 分类指数强弱评分、行业归属逻辑 |
| `backtest/run.py` | 回测指标计算、调仓日期生成 |
| `strategies/sieve.py` | 排名函数、选股逻辑 |
