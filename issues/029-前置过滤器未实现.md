## 问题描述

当前代码没有任何机制排除 ST 股票、退市风险警示、次新股和流动性极差的股票。`factor-design-v2.md` 已定义了完整的前置过滤器规则，但代码中对应模块 `gate/pre_filter.py` 未实现。

## 应实现规则

```
├─ ST / *ST 股票 → 直接排除
├─ 退市整理期股票 → 直接排除
├─ 上市不满60个交易日 → 排除
├─ 近20个交易日日均成交额 < 2000万 → 排除
└─ 近30个交易日连续跌停 → 排除
```

## 频率

- 周频：ST列表、退市状态、上市日期（触发后回溯已持仓）
- 日频：成交额、连续跌停（仅在候选池筛选时生效，不回溯已持仓）
