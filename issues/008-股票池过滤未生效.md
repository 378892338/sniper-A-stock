# 008 配置中的 ST/新股过滤逻辑未实现

**严重程度**: MEDIUM  
**文件**: `data/fetch.py:14-24`, `config/settings.py:24-26`  
**类型**: 功能缺失

## 问题描述

配置中声明了排除 ST 和上市不足 180 天的新股：

```python
EXCLUDE_ST = True
EXCLUDE_NEW_IPO = True
IPO_DAYS_MIN = 180
```

但 `fetch_stock_list()` 只做了简单映射，完全没有应用这些过滤：

```python
def fetch_stock_list() -> pd.DataFrame:
    df = ak.stock_zh_a_spot_em()
    df = df.rename(columns={...})
    return df[["symbol", "name", "total_mv", "float_mv"]]
```

## 修复建议

```python
def fetch_stock_list(exclude_st=True, exclude_new_ipo=True, ipo_days_min=180):
    df = ak.stock_zh_a_spot_em()
    if exclude_st:
        df = df[~df["名称"].str.contains("ST")]
    # 新股过滤需要上市日期字段
    return df
```

## 影响

ST 股票可能进入选股池，其异常波动会污染因子计算和回测结果。
