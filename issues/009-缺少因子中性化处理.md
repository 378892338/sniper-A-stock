# 009 因子未做行业和市值中性化

**严重程度**: MEDIUM  
**文件**: `strategies/sieve.py:44-49`, `models/factors.py:369-431`  
**类型**: 方法论缺陷

## 问题描述

因子评分没有对市值、行业等常见风险因子做中性化处理。这意味着：

- 选出的股票可能只是小市值股票（市值因子驱动）
- 行业集中度可能很高（某行业整体评分高）
- 因子的 alpha 可能被常见风险因子解释

## 修复建议

在 `SieveStrategy.score_single` 中增加中性化步骤：

```python
def neutralize(scores, market_cap, industry):
    """对市值和行业做横截面回归取残差"""
    import statsmodels.api as sm
    X = pd.get_dummies(industry).astype(float)
    X['log_mv'] = np.log(market_cap)
    X = sm.add_constant(X)
    model = sm.OLS(scores, X).fit()
    return model.resid
```

## 影响

策略选股可能过度暴露于市值和行业风险。
