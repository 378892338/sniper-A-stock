# 024 — 空仓期零收益污染回测指标

**严重程度**：MEDIUM  
**审查日期**：2026-05-02  
**状态**：待修复  
**文件**：`backtest/run.py:128-134`

## 问题描述

```python
portfolio_returns = pd.Series(0.0, index=pd.date_range(self.start, self.end, freq="B"))
```

回测从 2019-01-01 开始，第一个调仓日是 2019-01-31。约 22 个交易日的空仓期保持 `0.0`。

```python
# calc_metrics line 46
er = returns.dropna()  # 只去掉 NaN，不去掉 0.0
```

`0.0` 不是 `NaN`，不会被 `dropna()` 移除。22 个交易日的零收益被纳入累计收益、年化收益率、波动率等全部指标计算。

## 影响

- 年化收益率被轻微拉低（多出约一个月零收益）
- 年化波动率被拉低（零波动交易日稀释了整体波动）
- 夏普比率轻微偏高（分子分母同时被影响，但方向取决于实际情况）

## 修复方案

将初始值从 `0.0` 改为 `np.nan`：

```python
portfolio_returns = pd.Series(np.nan, index=pd.date_range(self.start, self.end, freq="B"))
```

这样 `dropna()` 会自然排除空仓期，只计算有实际持仓的期间的指标。
