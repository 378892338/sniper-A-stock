# 025 — 停牌股票导致组合权重稀释

**严重程度**：MEDIUM  
**审查日期**：2026-05-02  
**状态**：待修复  
**文件**：`backtest/run.py:154-182`

## 问题描述

```python
w = 1.0 / len(holdings)  # 等权
weights = {h: w for h in holdings}

for sym, wt in weights.items():
    if sym in daily_returns and ds in daily_returns[sym].index:
        rets.append(daily_returns[sym].loc[ds] * wt)
if rets:
    day_ret = sum(rets)
```

当持仓中有股票停牌时，该股票不参与当日收益计算，但其余股票权重保持 `1/N` 不变。例如：

- 持仓 30 只，3 只停牌
- 实际只有 27 只参与计算
- 权重合计 = `27/30 = 0.9`
- 组合当日收益被稀释约 10%

## 修复方案

在每日收益计算前，对参与计算的股票权重做重新归一化：

```python
rets = []
wts = []
for sym, wt in weights.items():
    if sym in daily_returns and ds in daily_returns[sym].index:
        rets.append(daily_returns[sym].loc[ds])
        wts.append(wt)
if wts:
    wts = np.array(wts) / np.sum(wts)  # 归一化
    day_ret = np.sum(np.array(rets) * wts)
```
