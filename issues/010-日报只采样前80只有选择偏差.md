# 010 日报股票池只取前 80 只（按代码排序），存在严重选择偏差

**严重程度**: LOW  
**文件**: `run_daily.py:111`  
**类型**: 功能限制

## 问题描述

```python
sample = symbols[:80]  # 本次采样: 80 只
```

`load_stock_pool` 返回的是沪深 300+中证 500 的成分股（约 800 只），但只取前 80 只。列表是 `set` 转 `list` 后按代码排序的，这意味着：

- 只有 000xxx 和 002xxx 开头的股票被选中
- 300xxx（创业板）、600xxx（沪主板）、688xxx（科创板）几乎完全遗漏
- 日报不具代表性

## 修复建议

1. 按代码前缀分层随机采样
2. 或者异步并发获取全量数据而非 `time.sleep(0.3)` 串行

```python
# 按前缀分层取样
prefixes = {}
for s in symbols:
    pref = '688' if s.startswith('688') else s[:3]
    prefixes.setdefault(pref, []).append(s)
sample = []
for stocks in prefixes.values():
    sample.extend(random.sample(stocks, min(15, len(stocks))))
```
