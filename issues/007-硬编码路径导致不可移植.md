# 007 多文件硬编码绝对路径 `D:/Projects/quant-system`

**严重程度**: MEDIUM  
**文件**: `run_daily.py:8`, `backtest_600251.py:2`, `backtest_cross_section.py:6`  
**类型**: 代码质量

## 问题描述

三个文件在开头硬编码了绝对路径：

```python
sys.path.insert(0, 'D:/Projects/quant-system')
```

换一台机器或改名目录后代码直接无法运行。

## 修复建议

统一使用 `pathlib` 的相对路径：

```python
sys.path.insert(0, str(Path(__file__).parent))
```

这与 `main.py` 中的做法一致。
