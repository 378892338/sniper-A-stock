# 002 L4 共振层周线映射 bug，共振分数永远为 0

**严重程度**: CRITICAL  
**文件**: `models/factors.py:300-306`  
**类型**: 逻辑错误

## 问题描述

`calc_resonance_score` 中将日线日期转成 `"%Y-%W"` 格式的字符串，然后去周线 DataFrame 的 index 里查找：

```python
wi = pd.Timestamp(di).strftime("%Y-%W")
if wi in weekly_bull.index:
```

但 `df_weekly` 的 index 是 `pd.Timestamp`（日期时间），不是字符串。`wi in weekly_bull.index` 永远返回 `False`。

这意味着 **L4 共振层从未生效，共振加分永远为 0**。

## 修复建议

```python
def calc_resonance_score(df_daily, df_weekly):
    if df_weekly.empty:
        return pd.Series(0.0, index=df_daily.index)
    
    weekly_ma20 = calc_ma(df_weekly["close"], 20)
    weekly_ma60 = calc_ma(df_weekly["close"], 60)
    weekly_bull = weekly_ma20 > weekly_ma60
    
    score = pd.Series(0.0, index=df_daily.index)
    for di in df_daily.index:
        # 找到该日对应的最近一周
        week_end = di + pd.Timedelta(days=(6 - di.weekday()))
        week_start = week_end - pd.Timedelta(days=4)
        # 在周线数据中找对应日期
        candidates = weekly_bull.loc[week_start:week_end]
        if len(candidates) > 0 and candidates.iloc[-1]:
            score[di] = 1.0
    return score
```

## 影响范围

技术面总分少了共振层加成（`resonance * 10`），对技术面评分有约 10% 的损失。
