# 数据归档说明

## 目录结构

```
outputs/
├── backtest_cache/        # 回测结果缓存（参数哈希 → parquet）
├── trade_logs/            # 交易日志（按月分片 parquet）
├── evaluations/           # 绩效评估报告（日/周/月频）
├── optuna_studies/        # Optuna 贝叶斯优化 study 持久化
├── online_learn/          # 在线学习器参数
├── params/                # 参数导出/导入
│   └── params_export.json
├── reports/               # 回测对比报告 / 年度分析
└── 数据归档说明.md         # 本文件

data/
└── raw/_cache/            # 原始数据缓存（parquet）
    ├── sw_industry_cons_*.parquet   # 申万行业分类
    ├── market_daily_*.parquet       # 指数日线缓存
    ├── etf_daily_*.parquet          # ETF日线缓存
    └── backtest/                    # 回测数据缓存
```

## 数据清单

| 数据源 | 位置 | 行数 | 时间范围 | 更新方式 |
|--------|------|------|---------|---------|
| 个股日线 | `daily_bars` 表 | ~780万 | 2000-2026 | 每日增量 |
| 行业对比 | `industry_compare` 表 | ~5.2万 | 2019-2026 | 每日 |
| 龙虎榜 | `dragon_tiger` 表 | ~11万 | 2019-2026 | 每日 |
| 季报 | `quarterly_financials` 表 | ~30万 | 2019-2026 | 按季 |
| 北向资金 | `northbound_flow` 表 | 1710天 | 2019-2026 | 每日 |
| 申万行业 | `sw_industry_cons_*.parquet` | 5195只 | 最新快照 | 月频 |
| **资金流向(proxy)** | `fund_flow` 表 | **389万行** | **2019-2026全覆盖** | **从 daily_bars 计算** |
| **强势股** | `hot_stocks` 表 | **4.7万行** | **2019-2026全覆盖** | **从 daily_bars 计算涨停** |

## 更新命令

```bash
# 一键更新全部信号数据
python -c "from sniper.signals.download_all import download_all_signals; from sniper.signals.store import SignalStore; download_all_signals(SignalStore())"
```

## 数据依赖关系

```
daily_bars (基础层, 必须有)
  ├── industry_compare (依赖行业分类)
  ├── dragon_tiger (独立, 实时)
  ├── northbound_flow (独立, 实时)
  ├── fund_flow (独立, 逐股查询)
  ├── quarterly_financials (独立, 财务报表)
  └── hot_stocks (独立, 涨停池)
```
